# Foundry Toolkit tour

Use this guide to orient yourself quickly with Foundry Toolkit in VS Code.

## What to look for

- model access and deployment views
- agent workflow support
- local prototyping helpers
- deployment guidance for your next demo step

## Suggested first pass

1. Open the project in VS Code.
2. Install the recommended extensions.
3. Review the samples and start with a tiny prompt or invocation path.
4. Move to a live agent or model flow once the local setup is stable.
