# Kindling GitHub Copilot instructions
This repo is a hackathon ideation kit for GitHub Copilot + Foundry Toolkit.

Always prefer the smallest demoable change that a team can explain in one sentence.

Use the right custom agent for the current intent (select from the agent picker in Chat):
- Ideation and framing: **kindling** or **idea-generator**
- Build planning: **implementation-plan**
- Demo storytelling and audience pitch: **demo-storyteller**

Keep the story practical, visible, and easy to explain in one sentence. Favor a human-centered flow over a broad feature list.
